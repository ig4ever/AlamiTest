## Installation
Run this command
```sh
npm install
```
Then, run this command for IOS
```sh
cd ios && pod install
```
## How To Run
### Android
Run this command
```sh
npm run android
```
or
```sh
npm run android-release
```
### IOS
Run this command
```sh
npm run ios
```
